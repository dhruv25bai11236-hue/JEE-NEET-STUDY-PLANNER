# JEE/NEET STUDY PLANNER
JEE/NEET Strategic Study Planner

•	Overview of the Project: 
The “JEE/NEET Strategic Study Planner” is a Python-based Command Line Interface (CLI) which is basically designed to assist students in their preparation for competitive entrance exams like JEE and NEET, so that they can ace their preparation. 

•	Features:
This project consists of following features:
1.	Exam-Specific Roadmaps: Provides content and book recommendations for JEE (Physics, Chemistry, Math) and NEET (Physics, Chemistry, Biology).
2.	Dynamic Countdown: Automatically calculates the exact number of days remaining until the exam date.
3.	Smart Schedule Generation: Creates a daily timetable with calculated study hours (6 to 12 hours) based on the exam date.
4.	Syllabus Allocation: Randomly shuffles and allocates specific chapters to specific days to ensure full syllabus coverage.
5.	Resource Hub: Displays lists of recommended textbooks and subject-specific preparation tips.
6.	Input Validation: Includes error handling for incorrect date formats or invalid menu selections.
7.	Motivational Elements: Displays success mantras and strategies for the final week of preparation.

•	Technologies/Tools Used:
Programming Language: Python 3.13.9
Libraries:
1.	datetime: For handling dates and calculating the exam countdown.
2.	random: Used to shuffle syllabus chapters and select random tips.
3.	math: Used for calculating study session blocks.
